# Snake Game
This is a simple implementation of the classic Snake game written in Python using the Pygame library.


Installation
Make sure you have Python and Pygame installed. You can download python from python.org. To install Pygame, you can use pip, the Python package manager. Depending on your operating system, open a terminal or command prompt and run the following prompt: pip install pygame.

Usage
Run the game by executing main.py.

Use the arrow keys to control the direction of the snake. Eat the apples to grow longer. The game ends when the snake collides with itself.

Customization
You can customize the game by modifying the following parameters in the config.py file:

width: Width of the game screen.
height: Height of the game screen.
cellSize: Size of each grid cell.
fps: Frames per second.
appleColor: Color of the apple.
snakeColor: Color of the snake.
backgroundColor: Color of the background
